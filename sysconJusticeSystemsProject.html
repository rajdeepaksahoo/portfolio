Project Title: Syscon Justice Systems Project

  •             This project is a comprehensive software solution developed for justice and public safety agencies. It encompasses various functionalities to assist in case management, offender tracking, and administrative tasks. The system provides a user-friendly interface and ensures secure access to sensitive information. Additionally, an admin panel is included to empower administrators in managing agency data effectively.

    • Technologies Used :

    • Java EE
    • Spring
    • Spring JDBC
    • Postgrsql
    • Spring Security
    • Angular
    • JUnit
    • Log4j
    • Docker
    • Docker Compose

    Key Features :

    • Case Management
    • Offender Tracking
    • Administrative Tasks
    • Secure Access
    • Reporting

    Responsibilities :

  • ✓ Developed backend endpoints using Spring (including Spring REST) to handle requests and responses.
  • ✓ Conducted thorough unit testing to ensure the reliability and functionality of the backend components.
  • ✓ Handled exceptions in REST endpoints to provide informative responses and ensure graceful error handling.
  • ✓ Tested and validated the functionality of REST endpoints using tools like Postman.
  • ✓ Assisted in managing and optimizing the PostgreSQL database, including schema design and data manipulation tasks.
