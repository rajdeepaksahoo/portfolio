Project Title: Ground Booking System

  •             The Ground Booking System is a web-based application designed to streamline the process of booking sports grounds or venues for various events or activities. It provides a platform for users to search for available grounds, make bookings, and manage their bookings. The system ensures secure access, robust authentication, and authorization mechanisms to protect sensitive data.

    • Technologies Used :

    • Java EE
    • Spring Boot
    • Data JPA
    • Postgrsql
    • Spring Security
    • JWT
    • Swagger
    • Docker
    • Kubernetes
    • JUnit
    • Mockito
    • Log4j
    • Docker
    • Docker Compose
    • JMeter

    Key Features :

    • Ground Search
    • Booking Management
    • User Authentication and Authorization
    • Ground Administration
    • Addmission History
    • User-Friendly Interface

    Responsibilities :

  • ✓ Developed backend functionalities using Spring Boot, including RESTful APIs for ground search, booking management, and administrative tasks.
  • ✓ Implemented a MultiMaven project structure to modularize the application and manage dependencies efficiently.
  • ✓ Implemented JWT-based authentication and authorization mechanisms using Spring Security, ensuring secure access to the application's resources.
  • ✓ Utilized Spring Data JPA for efficient data access and manipulation, including CRUD operations and complex queries on ground booking data.
  • ✓ Implemented Swagger for API documentation, enabling developers to understand and interact with the API endpoints easily.
  • ✓ Leveraged PostgreSQL as the backend database to store and manage ground booking information securely.
  • ✓ Utilized Log4j for logging to capture application events and troubleshoot issues effectively.
  • Collaborated with frontend developers to integrate frontend and backend components seamlessly.
