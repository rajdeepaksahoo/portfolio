Project Title: Human Resource Management System (HRMS) - Microservices Architecture

  •             The HRMS is a modern and scalable software solution designed using a microservices architecture. It facilitates various human resource management functions within organizations, providing flexibility, scalability, and resilience. The system consists of independent microservices, each responsible for specific HR tasks, such as employee management, leave management, payroll processing, and more. These microservices communicate with each other via well-defined APIs, ensuring modularity and ease of maintenance.

    • Spring Boot
    • Data JPA
    • Postgrsql
    • Spring Security
    • OAuth2
    • Netflix Eureka (Service Discovery)
    • Spring Cloud Gateway
    • Kafka
    • Docker
    • Kubernetes
    • JUnit
    • Mockito
    • Log4j
    • Docker
    • Docker Compose
    • JMeter

    • Microservices Architecture
    • Service Discovery
    • API Gateway
    • Secure Authentication Using OAuth2
    • Asynchronous Communication
    • Containerization
    • Orchestration Using Kubernetes
    • Distributed Tracing
    • User-Friendly Interface

    Responsibilities :

  • ✓ Developed using a microservices architecture to achieve scalability, resilience, and modularity.
  • ✓ Leveraged Spring Boot for rapid development and Spring Cloud for building cloud-native applications.
  • ✓ Integrated Netflix Eureka for service discovery and Spring Cloud Gateway for API routing and management.
  • ✓ Utilized Kafka for asynchronous communication between microservices, ensuring loose coupling and fault tolerance.
  • ✓ Containerized microservices using Docker, enabling easy deployment and scaling across different environments.
  • ✓ Orchestrated containerized microservices using Kubernetes, ensuring efficient resource management and automatic scaling.
  • ✓ Implemented secure authentication and authorization mechanisms using OAuth2 across microservices.
  • ✓ Developed an intuitive and responsive user interface using Angular, providing a seamless user experience.
