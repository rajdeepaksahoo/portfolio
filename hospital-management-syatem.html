Project Title: Hospital Management System Description

  •             Hospital Management System Description: This is a web application that allows hospitals to manage their patients, doctors, appointments, and medical records. The application has a user-friendly interface and provides secure access to sensitive information. The website also has an admin panel where administrators can manage hospital data, including doctors, patients, and appointments. Highlights: Designed and developed the entire web application using Java EE, Spring Boot and Data JPA. Integrated a MySQL database to store and retrieve data. Implemented user authentication and authorization to ensure the security of the application. Developed features to manage patients, doctors, appointments, and medical records. Developed an admin panel to enable administrators to manage hospital data.
